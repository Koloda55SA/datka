# 📚 Документация (дипломдук иш)

Бул папкада «Датка» веб-тиркемеси боюнча квалификациялык (дипломдук) иштин
документациясы жайгашкан.

## Файлдар

| Файл | Сүрөттөмө |
|------|-----------|
| `Датка_квалификациялык_иш.docx` | Даяр дипломдук иш (Word, кыргызча, Ош МУ форматында) |
| `generate_doc.py` | Документти автоматтык түзүүчү Python скрипт |

## Документтин түзүмү

1. **Титулдук бет** — Ош МУ форматы (министрлик, университет, факультет, кафедра, тема)
2. **Мазмуну** — беттердин номерлери менен
3. **Кириш сөз (КИРИШҮҮ)** — актуалдуулук, максат, милдеттер, объект, предмет, методдор, маани, түзүлүш
4. **I-БАП** — Веб-тиркемелерди иштеп чыгуунун теориялык негиздери
5. **II-БАП** — «Датка» веб-тиркемесин долбоорлоо
6. **III-БАП** — Тиркемени ишке ашыруу жана тесттөө
7. **Корутунду (КОРУТУНДУ)**
8. **Колдонулган адабияттардын тизмеси**

## Кайра түзүү (документти жаңыртуу)

```bash
cd docs
python generate_doc.py
# Чыгат: Датка_квалификациялык_иш.docx
```

> Формат: Times New Roman 14pt, саптар аралыгы 1.5, абзац 1.25 см,
> талаалар сол 3 см / оң 1.5 см / өйдө-ылдый 2 см.

## ⚠️ Толтуруу керек болгон жерлер

Титулдук бетте студенттин аты-жөнү, тайпасы, илимий жетекчиси жана
рецензенти үчүн бош орундар (`____`) калтырылган — аларды өзүңүздүн
маалыматыңыз менен толтуруңуз.
