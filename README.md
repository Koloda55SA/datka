# «Датка» — Сулуулук салонунун веб-тиркемеси

> Дипломдук (квалификациялык) иштин алкагында иштелип чыккан веб-тиркеме.
> **Фронтенд** толук функционалдуу, **бэкенд** — демонстрациялык (placeholder),
> ВПС/сервер сатып албастан иштетүү үчүн ылайыкталган.

## 📌 Долбоордун максаты

«Датка» сулуулук салону үчүн заманбап, мобилдик түзмөктөргө ыңгайлашкан веб-сайт
түзүү. Сайт аркылуу кардарлар:

- салондун **кызматтары** жана **баалары** менен тааныша алышат;
- **шеберлер** жана алардын тажрыйбасын көрө алышат;
- **онлайн жазылуу** калтыра алышат (кезек күтпөстөн);
- салондун **байланыш** маалыматын жана картадан ордун таба алышат.

## 🚀 Иштетүү (локалда)

### 1. Фронтенд (өзүнчө да иштейт)

Бэкенд иштебесе да, фронтенд `data.js` ичиндеги фоллбэк маалыматтар менен иштейт.
Жөн гана `frontend/index.html` файлын браузерден ачыңыз, же:

```bash
cd frontend
# Эгер Python орнотулган болсо, жөнөкөй сервер:
python -m http.server 5500
# андан соң браузерден: http://localhost:5500
```

### 2. Бэкенд (placeholder)

```bash
cd backend
npm install
npm start
# Сервер: http://localhost:4000
```

Бэкенд иштеп турса, фронтенд автоматтык түрдө андан маалымат алат.
Иштебесе — фоллбэккке өтөт (сайт баары бир толук көрүнөт).

## 🔌 API (placeholder)

| Метод | Маршрут | Сүрөттөмө |
|-------|---------|-----------|
| GET  | `/api/health`   | Сервердин абалы |
| GET  | `/api/services` | Кызматтар тизмеси |
| GET  | `/api/masters`  | Шеберлер тизмеси |
| GET  | `/api/reviews`  | Пикирлер тизмеси |
| POST | `/api/bookings` | Жаңы жазылуу кошуу |
| GET  | `/api/bookings` | Бардык жазылуулар (админ/демо) |

> ⚠️ **Эскертүү:** жазылуулар азыр сервердин эс тутумунда (in-memory) сакталат.
> Чыныгы өндүрүштө бул жерге маалымат базасы (PostgreSQL / MongoDB) туташтырылат.
